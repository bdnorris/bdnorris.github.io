Required

HTML/CSS Path

  1. Front-end Foundations (10 points)

    This course introduces you to you the basics of HTML and CSS. (10 points)

    Download Front-end Foundations Slides

  2. The Elements of Web Design (10 points)

    This course digs into the user-centered planning and workflow of building a site or app. A lot of the content here is based on the idea of user testing. (User testing is hard, and expensive, but if you can begin to better put yourself in the user’s place, you can user test yourself. You can also use colleagues and friends as cheap user testing.)

    Download The Elements of Web Design Slides

  3. Fundamentals of Design (5 points)

    This course talks about type, color, and layout in reference to web design. It’s geared towards non-designers, so much of this is things you already know, and probably have a better handle on than this course. However, there is a handful of useful content here that should be new to you, so I think it’s worth tackling. You will be annoyed.

  4. CSS Cross-Country 15 points)

    This course throws you into the deep-end of CSS. Remember to check MDN or cssreference.io for help!

    Download CSS Cross-Country Slides

  5. Front-end Formations (10 points)

    This course talks about some newer items in HTML/CSS. Of course, everything is new to you anyway, but the items this course talks about are important. Use MDN or the msuweb.design source code for help.

    Download Front-end Formations Slides

  6. Discovering Dev Tools [in the electives path] (5 points)

    This will walk you through the Chrome developer tools. These tools are essential for debugging your work. One caveat, chrome has already advanced since they made this course, so you may find some icons have changed. You will obviously need to use Chrome for this, even all modern browsers to have developer tools of their own. You only need to do sections 1, 2 and 5. The others refer mostly to javascript debugging. And feel free to skip any of the challenges, since this is a little out of date, some of them won’t work correctly.

  7. Journey into Mobile (15 points)

    1. This course walks you through how to design and develop websites for any device.
    2. Some notes: It’s not usually necessary to convert your pixels to EMs anymore, but EMs still have their benefits.
    3. This is where stuff starts to get extremely opinionated and arguments about best practices start. Pay attention to the technical aspects of media queries, etc. But I’ll definitely have things to say about “mobile first” and “breakpoints.”
    4. This course is also a little old, and there are some newer better options for things like responsive and retina images I’ll talk about in class.

    Download Journey Into Mobile Slides
