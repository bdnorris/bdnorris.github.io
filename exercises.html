E1: Document Markup

Points: 5

We're going to start with a quick review of document markup. I copied the contents of a webpage into a Thimble project and I'd like you to add appropriate HTML markup to it.

Here's the original article: The Moment CSS Started Making “Sense”

Here's the Thimble Project https://thimbleprojects.org/bdnorris/241251

Add appropriate markup to the document, including heading (h1, h2, etc.) tags, paragraph tags, lists, links, and any other divisions or structure you think is appropriate.

I've added any "meta" information inside square brackets ([]). If you see a picture or link in those, the idea is you should add an <img> tag or an <a> tag for the links. The images have already been uploaded.

Hit "Remix", then, when you're done, publish your page and send me the link via Slack.
